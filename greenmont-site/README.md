# Greenmont Luxury Site

Этот архив содержит структуру проекта для сайта Greenmont, подготовленную для загрузки в GitHub или Codex. Структура включает папки для медиа-ресурсов и исходные файлы. Перенесите сюда код компонента `GreenmontLuxurySite.jsx` из canvas, добавьте изображения в папку public/media в соответствии с файлом media_map, и затем вы можете загрузить проект в GitHub или использовать Codex для дальнейшей работы.

## Структура

- `src/GreenmontLuxurySite.jsx` — вставьте сюда код React-компонента.
- `src/App.jsx` — точка входа для React-приложения.
- `public/media/hero` — изображения для hero блока.
- `public/media/story` — изображения для секций истории.
- `public/media/apartments/shared` — изображения для апартаментов.
- `public/media/gallery` — изображения для галереи.

Файлы media_map и manifest которые были сгенерированы ранее приведены в отдельных файлах.
